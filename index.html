Sincronizar com Azure DevOps

Esta operação irá sincronizar os dados do dashboard em tempo real com a API do Azure DevOps.

Como gerar seu Personal Access Token (PAT):
  1. No Azure DevOps, clique no ícone de configurações de usuário (canto superior direito) e selecione Personal Access Tokens.
  2. Clique em New Token.
  3. Dê um nome (ex: Squad Fiscal Dashboard) e defina a expiração.
  4. Em Scopes, marque a opção Custom Defined.
  5. Role até a seção Work Items e marque a permissão Read (Leitura).
  6. Clique em Create e copie o token gerado.
